## Описание проекта
Данный проект представляет собой разработку CRM-системы для компании, занимающейся продажей обучающих курсов. 
Система предназначена для автоматизации процессов приема заявок, обработки сделок, управления продуктами и ценами, а также управления контактами клиентов.
## Основные функции
1. **Сбор Заявок**:
   - Клиенты оставляют заявки на сайте, которые автоматически обрабатываются и распределяются сотрудникам колл-центра.
2. **Обработка Сделок**:
   - Создание возможных сделок на основе согласия клиентов и автоматическое распределение задач на менеджеров по продажам.
3. **Управление Продуктами**:
   - Продукты оформляются с учетом различных параметров (формат проведения, предмет и т.д.) и возможности внесения скидок.
4. **Работа с Прайс-листами**:
   - Прайс-лист и его позиции управляются централизованно. Система обеспечивает автоматический расчет цен на основе введенных скидок.
5. **Мониторинг Статусов**:
   - Все этапы - от интереса до возможной сделки - отображаются в виде бизнес-процессов с возможностью отслеживания текущего статуса.
## Техническое задание
- Создание таблиц согласно требованиям.
- Реализация плагинов и процессов, включая маршрутизацию, создание записей и автоматический расчет цен.
## Дополнительно
- Рассмотрено дополнительное задание на улучшение структуры прайс-листа с возможностью создания нескольких версий и уникальных комбинаций позиций.
